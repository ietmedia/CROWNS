# Crowns Enchanted

Luxury holistic hair & wellness salon CRM and booking platform for Ashley's salon in Marietta, GA. Clients book online; the owner manages bookings, clients, staff, revenue, and AI-powered outreach from one admin panel.

**Stack:** Next.js 16 · InsForge (auth + DB + storage) · Stripe · Resend · Anthropic Claude · Telegram · WhatsApp · Google Calendar · PostHog · Tailwind CSS v4 · Framer Motion

## Local development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
